# DesktopConductor
## A personal desktop macropad to control media and audio playing on your computer.

#### DesktopConductor is one of my very first hardware projects and first time using HackClub softwars like lapse, hackatime, wakatime, and the stardance projects.
---
It consists of an oled screen which will in future iterations be adapted into a song display for the current song and artist, 3 keys for pause/play, back, and skip in media controll, and a rotary encoder which can be used to adjust the volume and pressed to mute and unmute.

Currently the firmware is basic and has not be made to include these features but they will be added and worked on after the prototype is first assembled and built.
